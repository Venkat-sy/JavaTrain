# Categorized DSA & LeetCode Problems

This repository contains a curated collection of Data Structures and Algorithms (DSA) and LeetCode problems. The solutions are organized systematically based on problem concepts and patterns.

## Categories

- **Arrays & Hashing**: Fundamentals of array manipulation and hash maps.
- **Strings**: String processing, anagrams, and substrings.
- **Dynamic Programming**: Recursion, memoization, and bottom-up DP concepts.
- **Intervals & Greedy**: Dealing with overlapping intervals and greedy choices.
- **Sliding Window, Two Pointers & Stacks**: Efficient subarray processing and linear scan techniques.
- **Design & Sorting**: Custom data structure design (like LRU Cache) and sorting algorithms.
- **Concurrency**: Multithreading problems.
- **Math & Bit Manipulation**: Mathematical approaches and bit-level optimizations.

All solutions are written in Java and were extracted to help structured learning and easy referencing!
